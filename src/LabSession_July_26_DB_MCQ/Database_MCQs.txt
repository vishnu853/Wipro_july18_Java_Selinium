1. What does JDBC stand for?
A. Java Distributed Connection
B. Java Database Connectivity
C. Java Database Compiler
D. Java Direct Connect

Answer: B. Java Database Connectivity

2. Which package contains JDBC classes and interfaces?
A. java.sql.*
B. javax.jdbc.*
C. java.db.*
D. org.jdbc.*

Answer: A. java.sql.*


3. Which class is used to make a connection with a database?
A. ConnectionManager
B. DataDriver
C. DriverManager
D. DBConnector

Answer: C. DriverManager


4. Which of the following is the correct JDBC URL for MySQL?
A. jdbc:mysql:localhost/db
B. jdbc:mysql//localhost:3306/db
C. jdbc:mysql://localhost:3306/dbname
D. jdbc:mysql\\localhost:3306/dbname

Answer: C. jdbc:mysql://localhost:3306/dbname


5. Which interface is used to execute SQL statements in JDBC?
A. Statement
B. Connection
C. ResultSet
D. PreparedStatement

Answer: A. Statement


6. What does executeUpdate() return?
A. Number of rows affected
B. Boolean value
C. A ResultSet object
D. Nothing

Answer: A. Number of rows affected


7. Which method is used to execute SELECT queries?
A. executeUpdate()
B. execute()
C. executeQuery()
D. runQuery()

Answer: C. executeQuery()


8. What does a ResultSet represent?
A. A SQL statement
B. A table of data from a database query
C. A type of exception
D. The number of rows affected

Answer: B. A table of data from a database query


9. Which of the following is true about PreparedStatement?
A. Cannot be reused
B. Helps prevent SQL injection
C. Does not support parameters

Answer: B. Helps prevent SQL injection


10. Which is the correct syntax to call a stored procedure using CallableStatement?
A. {execute procedure procedureName}
B. {call procedureName()}
C. exec procedureName()
D. call procedureName

Answer: B. {call procedureName()}



11. What is the purpose of Class.forName("com.mysql.cj.jdbc.Driver")?
A. Create a database
B. Load the JDBC driver class
C. Close the database
D. Register a user

Answer: B. Load the JDBC driver class

12. Which method is used to close a database connection?
A. shut()
B. terminate()
C. conn.close()
D. disconnect()

Answer: C. conn.close()


13. Which interface is used to represent the result of a SQL query?
A. ResultSet
B. SQLData
C. DBResult
D. ResultView

Answer: A. ResultSet

14. Which of the following is not a type of JDBC Statement?
A. SimpleStatement
B. PreparedStatement
C. CallableStatement
D. Statement

Answer: SimpleStatement


15. How are parameters set in a PreparedStatement?
A. Using set()
B. Using add()
C. Using put()
D. Using setInt(), setString(), etc.

Answer: D. Using setInt(), setString(), etc.
